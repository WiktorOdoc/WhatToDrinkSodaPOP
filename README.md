# WhatToDrinkSodaPOP
What should I drink? Soda POP edition. 

In src/main/resources directory:

questions.csv contains question data in format:

`QUESTION_ENUM,Actual question text`

drinks.csv contains drink data in format:

`DRINK_ENUM,Drink name,drink_file.png`

`/images/*.png` contain all the images which should be named according to drink_file.png field in drinks.csv file.

QUESTION_ENUM and DRINK_ENUM are directly refered to in the code so don't modify them.

Drools version is 7.46.0

Java version is 11

Eclipse version is 2021-9
